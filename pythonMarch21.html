JOB TITLE: Python Developer

  • Mar 20, 2021

Python is a general-purpose interpreted, interactive, object-oriented, and high-level programming language. It was created by Guido van Rossum during 1985- 1990. Like Perl, Python source code is also available under the GNU General Public License (GPL). This tutorial gives enough understanding on Python programming language.

OUR STACK :

Python, Django, MongoDB

JOB DESCRIPTION :

As a Python developer, the candidate will participate in the design / development / customization and maintenance of software applications. The candidate will also be maintaining a close working relationship with the other application stakeholders. Candidate must also demonstrate the ability to independently work in a fast paced environment requiring multitasking, efficient time management and excellent communication skills

RQUIRED QUALIFICATIONS AND SKILLS:

• 1+ years of development experience on Python

• 1+ years of development experience using Django, Numpy, Pandas, Matplotlib

• Django , HTML , CSS , Javascript , Numpy , Pandas , Matplotlib , Working on an No SQl and SQL databases, like Mongo, Dynamo, SQL, PGSQL respectively

• At least 1 years of experience in UI development and have varied portfolio of work to show for it

• Experience of developing applications using MVC architecture.

• Have experience of developing the multi-tenant Applications using Python+ and API development with micro service

• Experience on OOPS/Design Pattern and 3 tier architecture application.

• Experience on Mongodb , Mongodb Server and above.

• Experience of code review and code refactoring with best industry practices.

• Knowledge of software development life-cycle methodology e.g. Waterfall, Agile etc.

# Years of Experience:

Python Development experience: 1+ years
Django: 1+ years
